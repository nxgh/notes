
## HSLA
- 色相 Hue
- 饱和度 Saturation
- 亮度  Lightness
- 透明度 Alpha

| 值                | 描述                                               |
|------------------|--------------------------------------------------|
| hue - 色相         | 定义色相 (0 到 360) - 0 (或 360) 为红色, 120 为绿色, 240 为蓝色 |
| saturation - 饱和度 | 定义饱和度; 0% 为灰色， 100% 全色                           |
| lightness - 亮度   | 定义亮度 0% 为暗, 50% 为普通, 100% 为白                     |
| alpha - 透明度      | 定义透明度 0（透完全明） ~ 1（完全不透明）                         |
